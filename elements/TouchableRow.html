The basic RowCell for use within ListView. It displays
an image, primary text, secondary text, a value,
and an arrow right icon if the respective props are
specified.

Platforms: ios, android, web

Composes: TouchableRowCell, Image, Text

Props

Name Type Default Value Description
primaryText string|node
secondaryText string|node
value string|node A value attribute of the cell, to appear on the right
onPress func (required)
image object|node An thumbnail to be displayed on the left of the cell
height number
inverted bool
showMore bool Displays a right arrow on the right hand side of the cell
